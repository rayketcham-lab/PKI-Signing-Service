Get Started

Up and Running in 60 Seconds

If it takes longer than that, we owe you a coffee.

1
Install via Cargo
cargo install pki-sign
2
Import your PFX certificate
pki-sign import --pfx my-cert.pfx --label my-cert
3
Sign your first binary
pki-sign pe --cert my-cert --timestamp --file my-app.exe
